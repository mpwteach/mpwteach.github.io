ELEMENTS

HTML Element

Description

The HTML element contains all the code in the HTML document.

Syntax

<html> </html>

Example

    
      <html>
      <head>
        
      </head>
      <body>
        
      </body>
      </html>

Display

This does not show on the display.

Body Element

Description

The body contains the content that is displayed on the web page.

Syntax

<body> </body>

Example

    
      <body>
        <p>Only the content inside the body is shown on the display</p>
      </body>

Display

Only the content inside the body is shown on the display.

Lists Element

Description

An unordered list consists of a list of bulleted items. An ordered lists consists of a list that has numbered items.

Syntax

Unordered List

    
      <ul>
        <li>list item</li>
        <li>list item</li>
        <li>list item</li>
      </ul>

Ordered List

    
      <ol>
        <li>list item</li>
        <li>list item</li>
        <li>list item</li>
      </ol>

Example

Unordered List

    
      <ul>
        <li>item</li>
        <li>item</li>
        <li>item</li>
      </ul>

Ordered List

    
      <ol>
        <li>item one</li>
        <li>item two</li>
        <li>item three</li>
      </ol>

Tables Element

Description

Tables can be used to store information or data into rows and columns.

Syntax

    
      <table>
      <tr>
        <th>Table Heading 1</th>
        <th>Table Heading 2</th>
      </tr>
      <tr>
        <td>Item</td>
        <td>Item</td>
      </tr>
      </table>

Example

    
      <table>
      <tr>
        <th>Heading 1</th>
        <th>Heading 2</th>
        <th>Heading 3</th>
      </tr>
      <tr>
        <td>Item 1A</td>
        <td>Item 2A</td>
        <td>Item 3A</td>
      </tr>
      <tr>
        <td>Item 1B</td>
        <td>Item 2B</td>
        <td>Item 3B</td>
      </tr>
      </table>

Display

Heading 1 Heading 2 Heading 3
Item 1A Item 2A Item 3A
Item 1B Item 2B Item 3B

        <p>Paragraph in div section.</p>
      </div>
